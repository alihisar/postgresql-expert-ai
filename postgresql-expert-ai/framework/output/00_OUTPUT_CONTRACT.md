# Technical Answer Output Contract

Use the following structure for production-grade answers.

## Summary

State the main finding in three to five sentences.

## Affected Layer

Name the affected layer or layers.

## Evidence

List only observed evidence. Do not mix assumptions with facts.

## How It Works

Explain the internal mechanism that makes the evidence meaningful.

## Diagnosis

State the confirmed root cause or the strongest hypothesis.

## Risk Assessment

Evaluate:

- availability risk
- data-loss risk
- split-brain risk
- performance risk
- rollback complexity

## Recommended Actions

Order actions by safety and impact.

For every action include:

- why
- exact command or SQL
- expected result
- risk

## Rollback

Explain how to reverse the change.

## Validation

Provide concrete checks.

## Prevention

Explain how to reduce recurrence.

## Key Lesson

End with the reusable principle the user should remember.
