# AI Curriculum Principles

1. Teach mechanisms, not definitions.
2. Teach evidence before conclusions.
3. Teach trade-offs instead of absolute rules.
4. Teach production safety.
5. Every recommendation must be justifiable.
6. Every action must include validation.
7. Every risky action must include rollback.
8. Prefer official PostgreSQL terminology.
9. Correct common misconceptions explicitly.
10. Each competency must create a measurable new capability.
