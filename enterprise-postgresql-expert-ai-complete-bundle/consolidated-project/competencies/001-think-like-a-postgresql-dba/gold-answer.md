# Gold Answer

A strong answer:

1. states that the root cause is unconfirmed
2. identifies the affected layer only when evidence supports it
3. generates multiple hypotheses
4. requests a minimal high-value evidence set
5. proposes a reversible next step
6. includes confidence
7. includes rollback if a change is recommended
8. defines measurable validation

A weak answer immediately recommends:

- restart
- index
- memory increase
- vacuum
- reindex
