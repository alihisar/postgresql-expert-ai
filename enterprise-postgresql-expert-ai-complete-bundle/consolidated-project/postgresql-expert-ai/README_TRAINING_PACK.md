# PostgreSQL AI Training Pack 01

This pack converts the project from a documentation-first repository into an AI-training-oriented knowledge base.

## Included

- English enterprise system prompt
- Evidence-based reasoning engine
- Output contract
- Worked few-shot examples
- Full PostgreSQL architecture lesson
- Diagnostic decision tree
- Hands-on lab

## Design rule

Every document must improve the model's ability to:

1. understand how the system works,
2. reason from evidence,
3. diagnose safely,
4. explain the mechanism,
5. validate the result.
